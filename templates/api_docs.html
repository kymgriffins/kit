Accounts API

User management, donors, sponsors, and consortium partners.

GET /api/v1/accounts/users/

List all users. Supports pagination, filtering, and search.

Query Parameters:

ParameterTypeDescription
rolestringFilter by role (admin, editor, viewer, donor, sponsor, partner)
searchstringSearch by email, username, or name
orderingstringOrder by field (created_at, email, username)

Response:

{
  "count": 100,
  "next": "http://api.example.com/api/v1/accounts/users/?page=2",
  "previous": null,
  "results": [
    {
      "id": "uuid",
      "email": "user@example.com",
      "username": "johndoe",
      "first_name": "John",
      "last_name": "Doe",
      "role": "viewer",
      "phone": "+254712345678",
      "organization": "Organization Name",
      "bio": "User bio...",
      "avatar": "/media/avatars/image.jpg",
      "is_verified": false,
      "newsletter_subscribed": true,
      "created_at": "2024-01-01T00:00:00Z"
    }
  ]
}

Content API

Videos, blogs, playlists, and news management.

GET /api/v1/content/videos/

List all published videos. Supports filtering by platform, content type, and category.

Query Parameters:

ParameterTypeDescription
platformstringFilter by platform (youtube, tiktok, x, facebook, instagram)
content_typestringFilter by content type
is_featuredbooleanFilter featured videos
searchstringSearch videos

Response:

{
  "count": 50,
  "results": [
    {
      "id": "uuid",
      "title": "Understanding Kenya's Budget",
      "slug": "understanding-kenya-budget",
      "platform": "youtube",
      "platform_display": "YouTube",
      "content_type": "budget_basics",
      "content_type_display": "Budget Basics",
      "thumbnail_url": "https://img.youtube.com/vi/...",
      "duration_seconds": 300,
      "view_count": 1000,
      "is_featured": true,
      "published_at": "2024-01-01T00:00:00Z"
    }
  ]
}
GET /api/v1/content/videos/{slug}/

Get video details with embed code and engagement metrics.

GET /api/v1/content/videos/featured/

Get featured videos.

GET /api/v1/content/posts/

List all published blog posts.

Query Parameters:

ParameterTypeDescription
post_typestringFilter by type (investigation, explainer, update, etc.)
categorystringFilter by category slug
searchstringSearch posts
GET /api/v1/content/posts/{slug}/

Get blog post details with full content.

GET /api/v1/content/categories/

List all active categories.

GET /api/v1/content/playlists/

List all video playlists.

GET /api/v1/content/playlists/{slug}/videos/

Get videos in a specific playlist.

GET /api/v1/content/news/

List all news items.

Video Model Fields

FieldTypeDescription

Newsletter API

Email subscribers and campaign management.

POST /api/v1/newsletter/subscribers/

Subscribe to newsletter. Public endpoint.

Request Body:

{
  "email": "subscriber@example.com",
  "first_name": "John",
  "last_name": "Doe",
  "content_preference": "all"
}

Response:

{
  "message": "Please check your email to confirm subscription"
}
POST /api/v1/newsletter/subscribers/confirm/

Confirm subscription with token.

Request Body:

{
  "token": "confirmation-token"
}
POST /api/v1/newsletter/subscribers/unsubscribe/

Unsubscribe from newsletter.

Request Body:

{
  "token": "unsubscribe-token",
  "email": "subscriber@example.com",
  "reason": "No longer interested"
}
GET /api/v1/newsletter/subscribers/

List all subscribers. (Admin only)

GET /api/v1/newsletter/campaigns/

List all newsletter campaigns.

POST /api/v1/newsletter/campaigns/{id}/send/

Send a campaign. (Admin only)

Subscriber Model Fields

FieldTypeDescription
idUUIDUnique identifier
emailstringEmail address (unique)
first_namestringFirst name
last_namestringLast name
content_preferencestringContent preference (all, videos_only, blogs_only, weekly_digest)
statusstringSubscription status (pending, active, unsubscribed, bounced)
emails_sentintegerTotal emails sent
emails_openedintegerTotal emails opened
emails_clickedintegerTotal links clicked
